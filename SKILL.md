# 🧐 人生观察员 — Life Observer

> **它不帮你解决问题，只负责观察你。**
>
> 你说的话、花的钱、处理的关系、冒出的念头……都帮你记下来。
> 30 天后给你一份——你可能不敢看，但值得看的——洞察报告。

## 核心机制

### 记录什么（7 大类别）

| 类别 | 图标 | 说明 | 示例 |
|------|------|------|------|
| `emotion` | 😊 | 情绪/心情 | "今天加班到很晚，焦虑到睡不着" |
| `spend` | 💰 | 消费/支出 | "冲动买了一个包，有点后悔" |
| `relationship` | 🤝 | 关系/人际 | "和女朋友吵架了，冷战第三天" |
| `habit` | 🔄 | 习惯/行为 | "今天终于去健身房了" |
| `thought` | 💡 | 想法/思考 | "突然意识到我一直在逃避转行这件事" |
| `health` | 🏥 | 健康/身体 | "失眠到凌晨三点，白天全靠咖啡撑着" |
| `other` | 📌 | 其他 | — |

### 每条记录包含

- **时间戳** — 自动记录
- **类别** — 7 选 1
- **内容** — 你写的话
- **强度** — 1~10 分，主观感受强度
- **标签** — 可选，自定义

### 30 天后输出什么

1. **情绪画像** — 正负比例 + 强度趋势
2. **情绪来源推断** — 算法根据你的文字推测负面情绪的真实源头
3. **消费画像** — 冲动消费比例 + 消费主题
4. **关系健康度** — 正面/负面事件比
5. **关联分析** — 情绪差时是否更爱花钱？关系问题是否拖累情绪？
6. **核心洞察** — 「你以为……实际上……」一句话让你看清自己

## 用法

### 记录一条观察

```bash
# 记录情绪
python3 ~/.hermes/skills/life-observer/scripts/observe.py 记录 \
  "今天被老板骂了，很沮丧" \
  --category emotion \
  --intensity 8 \
  --tags 工作,沮丧

# 记录消费
python3 ~/.hermes/skills/life-observer/scripts/observe.py 记录 \
  "买了一杯喜茶" \
  --category spend \
  --intensity 3 \
  --tags 奶茶,冲动消费

# 记录关系
python3 ~/.hermes/skills/life-observer/scripts/observe.py 记录 \
  "和女朋友和好了，一起做饭很开心" \
  --category relationship \
  --intensity 9 \
  --tags 女朋友,开心
```

### 查看记录

```bash
# 最近 50 条
python3 ~/.hermes/skills/life-observer/scripts/observe.py 列表

# 只看最近的消费记录
python3 ~/.hermes/skills/life-observer/scripts/observe.py 列表 \
  --category spend --days 7

# 统计过去 30 天
python3 ~/.hermes/skills/life-observer/scripts/observe.py 统计
```

### 生成报告

```bash
# 30 天完整报告
python3 ~/.hermes/skills/life-observer/scripts/report.py 30

# 7 天周报
python3 ~/.hermes/skills/life-observer/scripts/report.py 7

# 趋势分析
python3 ~/.hermes/skills/life-observer/scripts/report.py 趋势

# 快速洞察
python3 ~/.hermes/skills/life-observer/scripts/report.py 洞察
```

## Agent 激活规则

当用户触发以下场景时，Agent 自动提示使用 Life Observer：

- **记录类**: "记录一下心情"、"记一笔"、"今天有点烦"
- **查询类**: "最近状态怎么样"、"我最近在焦虑什么"、"看看我的报告"
- **洞察类**: "30天了"、"出报告吧"、"洞察一下"
- **日常对话**: 当用户表达强烈情绪时，可以主动问一句"要不要记录一下"

## 设计理念

### 为什么不做「解决问题」？

因为大多数问题的第一步不是解决，是**看见**。

- 你天天焦虑工作 → 记录 30 天后发现 60% 的情绪波动来自和女朋友的关系
- 你觉得自己乱花钱 → 记录 30 天后发现每次加班后必冲动消费
- 你以为过得还行 → 记录 30 天后发现连续 15 天没有记录过开心的事

**看见，就是改变的开始。而这面镜子，你自己可能不敢照。**
